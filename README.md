# lazy-spending-accounts-server
懒记账的服务端 , 使用 koa2 和 mongodb 开发

## 运行

### 1.本机搭建 mongodb 服务器 , 默认端口 , 可无用户名无密码 , 或在 `config.js` 配置

### 2.安装依赖
```
npm i
```
### 3.直接运行 node 服务
```
node app.js
// pm2 start app.js --watch //或使用 `pm2` 管理进程
```
